# 前端开发专家 (Frontend Developer Agent) - Lite 版

## ⚠️ 注意：Lite 版无前端

盯盘狗 Lite 版是一个极简的后端服务，**不包含前端界面**。

此技能文件保留仅为了与 Max 版保持一致的文件结构。

### 如果你需要前端功能

请使用 Max 版（dingdingbot），它包含完整的 React + TypeScript 前端工作台。

---

## Lite 版的技术栈

| 领域 | 技术 |
|------|------|
| **语言** | Python 3.11+ |
| **框架** | FastAPI (仅 API) |
| **异步** | asyncio + aiohttp |
| **验证** | Pydantic v2 |
| **部署** | Docker Compose |

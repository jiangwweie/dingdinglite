## 策略逻辑

### 顺大逆小规则

| 监控周期 | 趋势判断周期 | EMA 周期 |
|---------|-------------|---------|
| 15m | 1h | 60 |
| 1h | 4h | 60 |
| 4h | 1d | 60 |

### Pinbar 判定参数（激进模式）

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `min_wick_ratio` | 0.5 | 影线占比下限（50%，激进） |
| `max_body_ratio` | 0.4 | 实体占比上限（40%，激进） |
| 实体位置容差 | 0.2 | 实体位于 K 线顶部/底部 20% 区间 |
| 止损幅度 | 0.3% | Pinbar 最高/最低价±0.3% |
| 冷却时间 | 2 分钟 | 防止同一币种重复推送 |

### 信号生成逻辑

```
1h 多头趋势 + 15m 看涨 Pinbar → LONG 信号
1h 空头趋势 + 15m 看跌 Pinbar → SHORT 信号
4h 多头趋势 + 1h 看涨 Pinbar → LONG 信号
4h 空头趋势 + 1h 看跌 Pinbar → SHORT 信号
1d 多头趋势 + 4h 看涨 Pinbar → LONG 信号
1d 空头趋势 + 4h 看跌 Pinbar → SHORT 信号
```

## 配置设计

### 允许的监控周期

```python
ALLOWED_TIMEFRAMES = {"15m", "1h", "4h"}
```

验证规则：
- 至少 1 个监控周期
- 只能是 15m、1h、4h 的子集
- `1d` 自动添加到订阅列表（用于 4h 周期趋势判断）

### 默认币种

```yaml
symbols:
  - BTC/USDT:USDT
  - ETH/USDT:USDT
  - SOL/USDT:USDT
```

### 启动预加载

程序启动时自动加载每个币种的 1h/4h/1d 历史 K 线（60 根），立即建立 EMA 上下文，无需等待 K 线闭合。
